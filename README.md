# Audio_To_Lines_Project
• Implemented Google Cloud Speech-to-Text API to convert audio content into text format, Designed an editable user interface component for users to enhance the accuracy of the transcribed content.

• Integrated SMTP Protocol within Spring Boot to facilitate sharing and receipt of refined audio transcription.
